# NoteVault

**A modern, offline-first notes application built with Angular 19**

Store your notes securely in your browser with zero server dependencies. Works completely offline using IndexedDB.

## Features

- ✅ Create, edit, and delete notes
- ✅ Offline storage using IndexedDB
- ✅ Auto-save notes while typing
- ✅ Search notes instantly
- ✅ Sort by created or modified date
- ✅ Dark / Light mode
- ✅ Export notes as JSON
- ✅ Responsive design for mobile and desktop
- ✅ Tag support for organizing notes

## Tech Stack

- Angular 19
- TypeScript
- IndexedDB
- RxJS
- SCSS

## Prerequisites

- Node.js installed
- npm installed
- Angular CLI installed
